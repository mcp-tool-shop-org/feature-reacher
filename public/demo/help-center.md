# Acme Analytics Help Center

## Getting Started

Welcome to Acme Analytics! This guide will help you set up your first dashboard.

### Quick Start

1. Install the tracking snippet on your website
2. Create your first dashboard
3. Add widgets to visualize your data

### Installing the Tracking Snippet

Copy and paste this code before the closing `</head>` tag:

```html
<script src="https://cdn.acme.io/analytics.js"></script>
```

## Dashboard Features

### Real-time Dashboard

View live visitor data as it happens. The Real-time Dashboard shows:
- Active users on site
- Current page views
- Live event stream

### Custom Alerts

Set up alerts to notify you when metrics cross thresholds. Alerts can be sent via:
- Email
- Slack
- Webhook

---

## Reports

### Scheduled Reports

Automate your reporting workflow:
1. Go to Reports > Scheduled
2. Select metrics and date range
3. Choose delivery frequency
4. Add recipient emails

### Export Options

Export data in multiple formats:
- CSV
- Excel
- PDF
- JSON (via API)

---

## Advanced Features

### Funnel Analysis

Track conversion through multi-step processes:
1. Define funnel steps
2. Set time window
3. Analyze drop-off points

### Cohort Analysis

Compare user groups over time based on acquisition date or behavior.

### Heatmaps

See where users click, scroll, and engage on your pages.

### Session Replay

Watch recordings of user sessions to understand behavior.

## Account Management

### Team Workspaces

Invite team members and assign roles:
- Admin: Full access
- Editor: Create and modify dashboards
- Viewer: Read-only access

### Multi-property Support

Manage multiple websites or apps from a single account.

## Troubleshooting

### Data Not Appearing

1. Verify tracking snippet is installed
2. Check for JavaScript errors
3. Ensure ad blockers aren't interfering

### Slow Dashboard Loading

Try reducing the date range or number of widgets.
